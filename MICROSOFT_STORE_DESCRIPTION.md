## Product Features (8 Main Points)

1. **Multiple AI Models & Providers** - Access various AI models from Xibe (default, no API key required), OpenAI (GPT-4, GPT-3.5, GPT-4 Turbo, GPT-4 Vision), Anthropic (Claude 3 Opus, Sonnet, Haiku), DeepSeek, Mistral, and custom API providers. Instant model switching with capability indicators, default model selection, and support for adding your own API endpoints.

2. **Advanced Chat & Conversation Features** - Real-time streaming responses, unlimited simultaneous conversations with auto-generated titles, rich Markdown support with code syntax highlighting, local SQLite database storage, persistent message history, time-based smart greetings, message search and navigation, chat renaming and deletion, and export functionality.

3. **Vision, Code Execution & Web Previews** - Image analysis with GPT-4 Vision, Gemini Vision, and Claude Vision models (charts, diagrams, screenshots, photos). Execute code securely in E2B Sandbox (Python, JavaScript/TypeScript, Java, R, Bash) with real-time results. Preview web components live with CodeSandbox (React, Vue, Angular, Svelte, HTML/CSS/JS) with multi-file project support and instant UI rendering.

4. **Long-Term Memory & AI Profiles** - Automatic memory extraction and storage across conversations (personal details, preferences, project context, feedback). Pre-built AI profiles (Socratic Tutor, Creative Writer, Strict Coder, Friendly Assistant, Technical Expert) plus custom profile creation with custom system prompts, profile management, and configurable memory limits.

5. **Advanced AI Capabilities** - Model Context Protocol (MCP) integration with multiple server support, tool and resource access, external data integration. Web search mode for real-time information with source citations. Reasoning mode (Think) for step-by-step problem solving, mathematical calculations, and logic puzzle analysis with detailed thought process visualization.

6. **Modern UI & Customization** - Material 3 Design with 5 themes (Dark, Light, Blue, Purple, Green), responsive desktop layout with sidebar chat history, smooth scrolling, professional code blocks, resizable windows, window management, keyboard shortcuts, and clean distraction-free interface. Comprehensive settings for model management, API key security, theme selection, system prompt customization, update channel selection, and MCP server configuration.

7. **Privacy, Security & Integration** - Local data storage with no cloud sync, encrypted API key storage, open source code (MIT License), no tracking or analytics, privacy-first approach. Deep link support with custom URL schemes, automatic updates with channel selection (Stable/Beta), and in-app donations via Razorpay (Cards, UPI, Net Banking, Wallets, multi-currency INR/USD).

8. **Cross-Platform & Free** - No API key required for default models, cross-platform support (Windows, macOS, Linux, Android, iOS), free forever, offline capable for saved conversations, fast and lightweight, Windows 10/11 optimized with native desktop experience, efficient resource usage, regular updates, and active development.
