üìö User Guide

Getting Started

  1. Launch Xibe Chat
  2. Start a new chat (automatic on first launch)
  3. Select an AI model from the model selector
  4. Type your message and press Enter/Send
  5. View real-time streaming responses

Key Concepts

  • Chat History: All chats are saved automatically
  • Model Selection: Switch models anytime during conversation
  • Image Input: Attach images for vision-capable models
  • Web Search: Enable for current information
  • Reasoning Mode: Get step-by-step explanations

üîå Custom Providers

Xibe Chat supports custom API providers, allowing you to use your own API keys with various AI services.

Built-in Providers

  • Xibe: Default provider with multiple models
  • OpenAI: GPT-4, GPT-3.5, and other OpenAI models
  • Anthropic: Claude 3.5 Sonnet, Haiku, and other models
  • OpenRouter: Access to 100+ models from various providers
  • Together AI: Open-source models with fast inference
  • Groq: Ultra-fast inference for open models

Adding a Provider

  1. Go to Settings ‚Üí Custom Providers
  2. Select a provider or add custom one
  3. Enter your API key
  4. Add models for the provider
  5. Start using in chat

Prerequisites

  • Flutter SDK (>= 3.0.0)
  • Dart SDK (>= 3.0.0)
  • IDE (VS Code, Android Studio, or IntelliJ)
  • Platform-specific tools (Xcode for macOS/iOS, Android Studio for Android)

Development Setup

# Clone repository
git clone https://github.com/yourusername/xibe-chat.git
cd xibe-chat

# Install dependencies
flutter pub get

# Run in debug mode
flutter run

# Hot reload: Press 'r' in terminal
# Hot restart: Press 'R' in terminal

Project Structure

  • lib/ - Main application code
  • lib/models/ - Data models
  • lib/providers/ - State management
  • lib/screens/ - UI screens
  • lib/services/ - API and services
  • lib/widgets/ - Reusable widgets
  • docs/ - Documentation
  • site/ - Documentation website

Building for Production

# Windows
flutter build windows --release

# macOS
flutter build macos --release

# Linux
flutter build linux --release

# Android
flutter build apk --release

# iOS
flutter build ios --release
