Use these examples to integrate Xibe Chat buttons into your website

1. Simple Button - Open New Chat with Message

Click the button to open Xibe Chat with a pre-filled message:

Open in Xibe Chat
<a href="xibechat://new?message=Hello%20World">Open in Xibe Chat</a>

2. Button with HTTPS Fallback

This button works even if the custom scheme isn't registered:

Open in Xibe Chat (HTTPS)
<a href="https://chat.xibe.app/app/new?message=Hello%20World">Open in Xibe Chat</a>

3. JavaScript Function (Recommended)

This function tries the custom scheme first, then falls back to HTTPS:

function openXibeChat(message) {
  const encoded = encodeURIComponent(message);
  const customScheme = `xibechat://new?message=${encoded}`;
  const httpsLink = `https://chat.xibe.app/app/new?message=${encoded}`;
  window.location.href = customScheme;
  setTimeout(() => {
    if (document.visibilityState === 'visible') {
      window.location.href = httpsLink;
    }
  }, 500);
}

4. Dynamic Message from Input

Enter a message and click the button:

function openWithInput() {
  const message = document.getElementById('messageInput').value;
  if (message) {
    openXibeChat(message);
  }
}

5. Empty Chat (No Pre-filled Message)

Open a new empty chat:

Start New Chat
<a href="xibechat://new">Start New Chat</a>

6. Alternative Query Parameter Names

You can use message, text, or prompt:

Using 'text' Parameter Using 'prompt' Parameter
xibechat://new?message=Hello
xibechat://new?text=Hello
xibechat://new?prompt=Hello
